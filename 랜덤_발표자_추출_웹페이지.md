# 랜덤 발표자 추출 웹페이지

## 개요

수업시간에 발표자를 랜덤으로 뽑아서 보여주는 웹페이지

---

## 목표 및 대상

초등학생 

---

## 핵심 기능

- 발표자를 랜덤으로 뽑아주는 기능
  - 추출 인원을 선택할 수 있는 기능
- 학생을 중간에 추가하는 기능 혹은 중간에 학생을 삭제하는 기능
  - 중간에 불출석 한 학생을 빼거나 늦게 온 학생을 다시 추가하는 기능
  - 한명씩 혹은 여러명을 일괄로 추가하는 기능
- 교사가 몰래 발표자를 미리 지정할  수 있거나 발표자 순서를 정하는 기능

---

## 화면 구성


발표자를 보여줄때 슬롯 머신 처럼 재미있게 보여주기
